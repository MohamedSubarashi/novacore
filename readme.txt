NovaCore is a premium multi-purpose WordPress theme designed for performance, SEO, and unlimited customization. Suitable for blogs, businesses, portfolios, SaaS, agencies, eCommerce, news websites, and landing pages.

== Description ==

NovaCore is a powerful and flexible multi-purpose WordPress theme built for modern websites. It combines cutting-edge performance with extensive customization options to create any type of website.

= Key Features =

* **Drag & Drop Builder** – Header and Footer builder with intuitive controls
* **Mega Menu Support** – Advanced navigation with mega menu capabilities
* **Dark Mode** – Built-in dark mode with automatic color scheme switching
* **Gutenberg Optimized** – Full block editor support with custom blocks and theme.json
* **WooCommerce Ready** – Complete eCommerce styling for shop, cart, and checkout
* **Custom Widgets** – Recent Posts, Social Counter, Newsletter, Author Card, Post Slider, Ad widgets
* **Advertisement System** – 6 ad placement areas with Customizer controls
* **Performance Optimized** – Minimal dependencies, efficient CSS, and lazy loading
* **SEO Friendly** – Semantic HTML5 markup and schema.org integration
* **Translation Ready** – POT file included for easy localization
* **RTL Support** – Full right-to-left language support
* **bbPress & BuddyPress** – Community and forum styling included
* **AI-Powered** – Built-in AI tools for content generation and optimization

= Custom Blocks =

NovaCore includes 11 custom Gutenberg blocks:
* Hero Block – Full-width hero sections with overlays
* Slider/Carousel – Responsive image and content sliders
* Post Grid – Filterable grid layouts for posts
* Feature Grid – Feature showcase with icons
* Buttons – Advanced button groups
* Team – Team member profiles
* Counter – Animated number counters
* CTA – Call-to-action sections
* FAQ – Accordion FAQ sections
* Testimonials – Client testimonial carousels
* Pricing – Pricing table columns

= Layouts =

* Home: Float-based layout with content (65%) and sidebar (34%)
* Blog: Standard loop with sidebar options

* Landing Page: Minimal header/footer for conversion pages

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and choose the novacore.zip file.
3. Click Install Now and then Activate.
4. Go to Appearance > NovaCore to configure theme settings.
5. Go to Appearance > Demo Importer to import demo content (requires WordPress Importer plugin).

== Frequently Asked Questions ==

= Does NovaCore work with page builders? =

Yes. NovaCore is compatible with major page builders including Elementor and Block Editor. It also includes blank and landing page templates.

= Is NovaCore translation ready? =

Yes. All theme strings are translatable via the /languages directory. A .pot file is included.

= Does NovaCore support WooCommerce? =

Yes. NovaCore includes complete WooCommerce styling for shop pages, product pages, cart, checkout, and my account pages.

= How do I set up the header image? =

Go to Appearance > Customize > Header Image to upload your logo/brand image (recommended size: 175x100px). When set, it replaces the site name in the header.

= How do I manage advertisement placements? =

Go to Appearance > Customize > NovaCore Ads to configure ad codes for all 6 placement areas.
