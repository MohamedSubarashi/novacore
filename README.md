# NovaCore WordPress Theme

A premium multi-purpose WordPress theme designed for performance, SEO, and unlimited customization.

## Features

- **Drag & Drop Builder** – Header and Footer builder with intuitive controls
- **Mega Menu Support** – Advanced navigation with mega menu capabilities
- **Dark Mode** – Built-in dark mode with automatic color scheme switching
- **Gutenberg Optimized** – Full block editor support with 11 custom blocks
- **WooCommerce Ready** – Complete eCommerce styling for shop, cart, and checkout
- **Custom Widgets** – Recent Posts, Social Counter, Newsletter, Author Card, Post Slider, Ad widgets
- **Customizable Homepage Slideshow** – Configure source (recent posts or specific category), autoplay, speed, and slide count via Customizer
- **Video Post Format** – Support for oEmbed (YouTube/Vimeo) and self-hosted MP4 videos
- **Advertisement System** – 6 ad placement areas with Customizer controls
- **Performance Optimized** – Minimal dependencies, efficient CSS, and lazy loading
- **SEO Friendly** – Semantic HTML5 markup and schema.org integration
- **Translation Ready** – POT file included for easy localization
- **RTL Support** – Full right-to-left language support

## Custom Blocks

- Hero, Slider/Carousel, Post Grid, Feature Grid, Buttons, Team, Counter, CTA, FAQ, Testimonials, Pricing

## Requirements

- WordPress 6.2+
- PHP 7.4+

## Installation

1. Download the theme zip file.
2. In your WordPress admin, go to **Appearance > Themes** and click **Add New**.
3. Click **Upload Theme** and choose the `novacore.zip` file.
4. Click **Install Now** then **Activate**.
5. Go to **Appearance > Customize** to configure theme settings.
